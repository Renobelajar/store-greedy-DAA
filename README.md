# Store Greedy DAA

Greedy algorithm implementation project for solving optimization problems in Design and Analysis of Algorithms (DAA).

## Files
- beli_cokelat.cpp
- jamuan_makan.cpp

## Features
- Greedy algorithm implementation
- Optimization problem solving
- C++ algorithm simulation
- Store and food distribution case studies

## Technologies
- C++
- Greedy Algorithm
- DAA Concept

## Problem Cases

### 1. beli_cokelat.cpp
Program for solving chocolate purchasing optimization problems using greedy methods.

### 2. jamuan_makan.cpp
Program for solving meal distribution or food-serving optimization problems using greedy algorithms.

## Concepts Used
- Greedy Algorithm
- Optimization
- Problem Solving
- Algorithm Analysis
